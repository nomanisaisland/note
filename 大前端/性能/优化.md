# 界面优化

主体最好使用flex布局，上中下间距平均开,结构模块一定要上中下分明，这样可以更好地适配长屏幕手机

```css
.box {
    display: flex;
    flex-direction: column;
    justify-content: space-between;
    align-items: center;
}
```
