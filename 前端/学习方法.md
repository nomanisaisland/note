# 学习新技术的第一大步：认知、了解阶段 #
<p>1.了解该技术的百科定义</p>
<p>2.该技术解决了什么问题</p>
<p>3.摸清楚同类的技术有哪些</p>
<p>4.了解该技术的主要组成部分</p>
<p>5.了解该技术出现的原因</p>
